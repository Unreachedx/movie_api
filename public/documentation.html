Movie App API Documentation

Welcome to the Movie App API documentation. This API allows you to manage users and access information about movies.

Base URL

The base URL for accessing the API is:

https://myflixapplication-paddy-fac687c8aed3.herokuapp.com

Note: Authentication is required to access the endpoints. Please make sure you are logged in to access the API.

Endpoints

Endpoint Method Description
/users POST Create a new user
/users GET Retrieve a list of all users
/users/:Username GET Retrieve a user by username
/users/:Username PUT Update an existing user by username
/users/:Username DELETE Delete a user by username
/users/:Username/favorites/:movieID POST Add a movie to a user's list of favorites
/users/:Username/favorites/:movieID DELETE Remove a movie from a user's list of favorites
/movies GET Retrieve a list of all movies
/movies/:title GET Retrieve a movie by its title
/movies/genre/:genre GET Retrieve movies by genre
/directors/:directorName GET Retrieve movies by director's name

Users Endpoints

  • Create User 
    POST /users

    Create a new user.

    Request Body:

    {
  "Username": "johndoe",
  "Password": "password123",
  "Email": "johndoe@example.com",
  "Birthday": "1990-01-01"
}
  • Get All Users 
    GET /users

    Retrieve a list of all users.

  • Get User by Username 
    GET /users/:Username

    Retrieve a user by username.

  • Update User 
    PUT /users/:Username

    Update an existing user by username.

    Request Body:

    {
  "Username": "newusername",
  "Password": "newpassword123",
  "Email": "newemail@example.com",
  "Birthday": "1991-02-02"
}
  • Delete User 
    DELETE /users/:Username

    Delete a user by username.

  • Add Favorite Movie to User 
    POST /users/:Username/favorites/:movieID

    Add a favorite movie to a user's list.

  • Remove Favorite Movie from User 
    DELETE /users/:Username/favorites/:movieID

    Remove a favorite movie from a user's list.

Movies Endpoints

  • List All Movies 
    GET /movies

    Retrieve a list of all movies.

  • Get Movie by Title 
    GET /movies/:title

    Retrieve a movie by its title.

  • Get Movies by Genre 
    GET /movies/genre/:genre

    Retrieve movies by genre.

  • Get Movies by Director 
    GET /directors/:directorName

    Retrieve movies by director's name.
